# 免费领牛奶引流转化功能说明

## 功能概述

免费领牛奶是一个会员转化功能，通过「打卡攒积分 → 解锁福利 → 开通会员 → 领取实物」的闭环路径，提升用户留存和会员转化率。

## 二、解锁与倒计时规则

### 2.1 解锁门槛
- **固定积分要求**：累计 100 积分
- **积分统计规则**：
  - 仅统计历史总获得积分（`totalEarnedPoints`）
  - 积分消耗（兑换礼品）不影响累计值
  - 与当前可用积分（`points`）独立计算

### 2.2 倒计时计算
```typescript
const UNLOCK_THRESHOLD = 100;  // 解锁门槛
const DAILY_POINTS_LIMIT = 50; // 单日积分上限

const remainingPoints = Math.max(0, UNLOCK_THRESHOLD - totalEarnedPoints);
const daysRemaining = Math.ceil(remainingPoints / DAILY_POINTS_LIMIT);
```

**示例：**
- 当前累计 0 积分 → 还有 2 天（100/50 = 2）
- 当前累计 25 积分 → 还有 2 天（75/50 = 1.5 向上取整）
- 当前累计 85 积分 → 还有 1 天（15/50 = 0.3 向上取整）
- 当前累计 100 积分 → 立即解锁

### 2.3 状态切换
积分达到 100 后，横幅自动从"未达标状态"切换为"达标状态"，无需刷新页面。

## 三、会员签约流程

### 3.1 新用户签约（未签约过会员）

**步骤1：点击横幅按钮**
- 弹出会员签约弹窗
- 显示「0元开通3天会员 免费领10枚鸡蛋」

**步骤2：查看会员权益**
- 打卡积分 ×1.2 倍
- 专属 AI 语音包
- 无限次健康咨询
- 会员专属礼品

**步骤3：确认自动续费规则**
- 前 3 天完全免费，无任何费用
- 免费期到期前 24 小时推送微信提醒
- 到期后自动按 29 元/月续费
- 可随时在「我的-会员管理」取消

**步骤4：勾选协议**
- 必须勾选《会员服务协议》和《自动续费代扣协议》
- 否则无法继续签约

**步骤5：确认签约**
- 点击「0元开通并领取」按钮
- 模拟微信代扣授权（实际需接入微信支付）
- 授权成功后开通 3 天免费会员

**步骤6：填写收货地址**
- 自动跳转至地址填写页面
- 填写收货人姓名、手机号、详细地址
- 提交后生成兑换订单

**步骤7：完成**
- 提示「牛奶兑换成功！将在7个工作日内包邮发货」
- 物流信息同步至积分商城的兑换记录
- 关闭弹窗，返回打卡页面

### 3.2 老用户签约（已签约过会员）

- 按钮文案改为「8元首月开通会员 领免费牛奶」
- 无免费期，直接按 8 元首月开通
- 其他流程与新用户相同

### 3.3 已开通会员

- 横幅显示「会员专属福利」入口
- 点击跳转至会员专属页面（待实现）

## 四、牛奶领取规则

### 4.1 领取资格
- 签约成功后立即发放兑换资格
- 单用户仅可领取 1 次（`hasClaimedMilk` 标记）
- 不可重复领取

### 4.2 配送规则
- 7 个工作日内包邮发货
- 仅支持中国大陆地区配送
- 物流信息同步至「积分商城-兑换记录」

### 4.3 取消续费影响
- 用户在免费期内取消自动续费
- **不影响**已发放的牛奶兑换资格
- 牛奶仍会正常发货

## 五、合规与风控

### 5.1 协议展示
- 签约页必须展示《会员服务协议》《自动续费代扣协议》入口
- 用户必须勾选同意后才可签约
- 协议内容需清晰告知续费规则

### 5.2 提醒机制
- 免费期到期前 24 小时推送微信服务通知
- 通知内容：「您的免费会员即将到期，明日将自动续费 29 元/月」
- 提供一键取消续费入口

### 5.3 投诉处理
- 用户投诉未明确告知自动续费
- 支持后台一键操作：
  - 取消会员
  - 全额退款
  - 关闭自动续费

### 5.4 防刷机制
- 同一设备仅可领取 1 次牛奶
- 同一微信账号仅可领取 1 次牛奶
- 检测到异常账号立即冻结兑换资格

## 八、使用说明

### 8.1 测试流程

1. 进入「健行」Tab，查看「健康打卡」标签页
2. 顶部显示免费领牛奶横幅，默认显示「还有 2 天即可领取」
3. 完成打卡任务，每完成一个任务，`totalEarnedPoints` 增加

5. 点击「立即领取免费牛奶」按钮
6. 查看会员签约弹窗，勾选协议
7. 点击「0元开通并领取」，模拟签约
8. 填写收货地址并提交
9. 查看成功提示，关闭弹窗
