# 健行系统使用指南

## 功能概览

健行系统包含以下核心模块：

### 1. 健康打卡模块
- ✅ 自动任务生成（基于 AI 健康建议）
- ✅ 手动添加任务
- ✅ 照片验证打卡
- ✅ 积分奖励系统
- ✅ 连续打卡统计

### 2. 积分商城模块
- ✅ 虚拟权益兑换（会员卡、语音包等）
- ✅ 健康周边兑换（茶包、按摩仪等）
- ✅ 实物礼品兑换
- ✅ 兑换记录查询
- ✅ 收货地址管理

### 3. 积分获取模块
- ✅ 打卡获取积分
- ✅ 分享内容获取积分
- ✅ 邀请好友获取积分

## 访问路径

用户可通过底部导航栏的「健行」Tab 进入系统，导航位置：

## 使用流程示例

### 场景 1：从 AI 建议创建打卡任务

1. 用户在首页使用任意健康工具（如睡眠分析）
2. 获取 AI 健康建议
3. 在建议底部显示 `AIAdviceCard` 组件
4. 点击「接受建议并加入打卡」按钮
5. 系统自动提取可执行任务并加入打卡列表
6. 跳转至打卡页面

**示例代码：**
```tsx
import { AIAdviceCard } from "../components/checkin/AIAdviceCard";

// 在健康工具结果页面中使用
<AIAdviceCard 
  advice={aiGeneratedAdvice}
  toolName="睡眠分析"
/>
```

### 场景 2：手动添加打卡任务

1. 进入打卡页面
2. 点击「添加任务」按钮
3. 输入任务内容（1-20字）
4. 确认添加
5. 任务出现在待完成列表

### 场景 3：完成打卡任务

**普通任务：**
1. 点击任务卡片的「完成」按钮
2. 积分即时到账
3. 任务移至已完成列表

**验证任务（需照片）：**
1. 点击任务卡片的「完成」按钮
2. 弹出拍照验证界面
3. 上传照片
4. 系统自动审核（模拟 1.5 秒）
5. 审核通过后积分到账

### 场景 4：兑换礼品

1. 切换至「积分商城」标签
2. 浏览礼品列表
3. 点击目标礼品的「兑换」按钮
4. 确认兑换信息
5. 实物礼品需填写收货地址
6. 确认后扣除积分，生成订单

### 场景 5：邀请好友

1. 切换至「邀请赚积分」标签
2. 复制邀请码或点击「分享给好友」
3. 好友使用邀请码注册
4. 双方同时获得积分

### 场景 6：分享内容获取积分

**在任意支持分享的页面（聊天记录、播客、社区帖子）：**

```tsx
import { ShareButton } from "../components/checkin/ShareButton";

// 添加分享按钮
<ShareButton 
  content={contentToShare}
  title="分享标题"
  contentType="podcast"
  className="px-4 py-2"
/>
```

## 积分规则

### 获取积分
- 普通任务打卡：1 积分/条
- 验证任务打卡：3 积分/条
- 会员打卡加成：×1.2 倍
- 分享内容：2 积分/次（同一内容每日限 1 次）
- 邀请好友：50 积分/人
- 被邀请注册：30 积分

### 连续打卡奖励
- 7 天：20 积分
- 30 天：100 积分
- 90 天：500 积分

### 限制条件
- 单日打卡积分上限：50 积分
- 单日分享积分上限：10 积分
- 单日自动任务上限：5 条
- 单日手动任务上限：3 条
- 单日总任务上限：8 条

## 风控机制

### 照片验证
- 同一照片不可重复使用（MD5 校验）
- 照片必须清晰可辨
- 格式限制：JPG、PNG
- 大小限制：≤10MB

### 异常行为检测
- 短时间内大量分享
- 同一设备多账号注册
- 同一 IP 单日注册超限
- 检测到异常行为将冻结积分
